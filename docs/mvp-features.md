# Oaiko - MVP機能仕様

## 概要

**Oaiko（おあいこ）** - 共有家計簿アプリのMVP（Minimum Viable Product）機能仕様。

- **名前の由来**: お相子 = 精算して貸し借りなし

---

## 機能一覧

### 1. 支出記録

| 項目     | 詳細                           |
| -------- | ------------------------------ |
| 金額     | 必須、数値入力                 |
| カテゴリ | 必須、プリセット or カスタム   |
| 支払者   | 必須、グループメンバーから選択 |
| 日付     | 必須、デフォルトは当日         |
| メモ     | 任意、自由入力                 |

### 2. 負担方法

支出ごとに負担方法を選択可能。

| 方法         | 説明           | 例（1000円、A/B/C 3人）                   |
| ------------ | -------------- | ----------------------------------------- |
| 均等         | 人数で等分     | A:334, B:333, C:333                       |
| 傾斜（割合） | 割合を指定     | A:50%, B:30%, C:20% → A:500, B:300, C:200 |
| 傾斜（金額） | 金額を直接指定 | A:500, B:300, C:200                       |
| 全額負担     | 1人が全額負担  | A:1000, B:0, C:0                          |

### 3. グループ・メンバー

- Nグループ対応（ユーザーは複数グループに所属可能）
- M人メンバー対応（グループごとに複数メンバー）
- グループ招待機能

### 4. カテゴリ

#### プリセット（案）

- 食費
- 日用品
- 光熱費
- 交通費
- 娯楽
- 医療費
- その他

#### カスタムカテゴリ

- グループごとにカスタムカテゴリを追加可能

### 5. 精算

- **月ごと表示**: 各月の精算額を表示
- **精算ステータス**: 精算済み / 未精算を管理
- **精算額計算**: 誰が誰にいくら払うかを自動計算

### 6. 分析・グラフ

- **カテゴリ別円グラフ**: 支出のカテゴリ別内訳
- **月別推移グラフ**: 支出の月別推移

### 7. 買い物リスト

| 機能         | 詳細                                         |
| ------------ | -------------------------------------------- |
| リスト共有   | グループメンバー全員で共有                   |
| アイテム追加 | 誰でも追加可能                               |
| 支出連携     | 購入時にチェック → 支出登録（合計金額入力）  |
| 自動削除     | 購入済みアイテムはリストから消える           |
| 履歴表示     | 履歴モードで過去の購入済みアイテムを確認可能 |
